乙未年九月十六與攸寧於戌時

tydqq Post in 未分类，
28 十月 2015
0

我有點自責
我沒有盡力勤奮
雖然上次師傅誇我了
可是我自己知道那次我沒有盡力勤奮
分明不夠勤奮，所以被師傅誇了，卻很羞愧
我在想 ，是不是師父其實看出來我沒練到盡善盡美…還是說我真的那兩個指法不錯

嗯 ，可是惰性難改也

約莫一月了，我新結實一女子
初見面是在師父那兒，我著一身黑色道袍，她一襲白帶淺綠漢服
然後，都是很靦腆的樣子。
也很柔弱
後來有一起爬過山，聊過天，參加茶會…
這個女子，也參佛
終，也欲問我修道
對其映像深刻是因她的執念令我震撼莫名
她追求一男子十一年未果，曾有一段在一起的經歷卻也終分飛
典型的爲情而傷，問佛寬心。忽見道童自在表象，羡慕莫名，便詢道途。
然後我又想到友人中另一個參佛，卻在接觸我后欲問道的人，你
在今天前，我又去看了些故事。好像，女子參佛總有一名爲“傷”的因。這傷多與情搭著邊，禪心遇上了情，這場修好似都凄美了些許
是如此啊。。。
我昨天看了達賴喇嘛六世的故事，也是引了我一陣唏噓
可是，我沒聽聞過有修佛得到真自在大解脫的受傷者
我心緒有些複雜的便是因爲這點了
我未見有修佛得到真自在大解脫的受傷者

不是受傷，修佛自然安然，也自然。
若是受傷呢，青燈盞，淺念佛諧.得到的是真自在還是另藏倔强的寂寞
然而我卻是無處得知了，畢竟我自己的道途都尚淺顯。
師父說我的一大弱點，想太多。我確實是在莫名多想，不過真的有些擔心與自己有所交集的佛者(因傷入道的)
也許這種爲他人擔憂的感叫做慈悲？

你和她，都在參佛的路途。都是途中遇見了看似自在安然的小道童，都有了問道的心思。
于道，我知道每一次窠臼艱難過後都是升華。但是佛，好像衹有至深處才是解脫，那種深度要多久去修又是否修的到，而在這之前的路途該是何等的酸楚
哈哈 ~ 瞭然了
佛和道…道中有一途曰佛，佛至深處可通道
佛也有道次第
出了道教詞匯這于心的牢，究其竟是老子所説的無名，强曰爲“道”
道是每個人的修行，衆生萬千各有萬千的道，它無名，無相，無形，無始無終，因存而在，這東西無名，“道”衹是老子強賦的名。。所以，我在想佛不也是麽
而且“道”可以包容“佛”的一切理，而佛卻強固定在了“佛”字上（當然了，道教的道也是強固定在了所謂道教之道上）

你説一切法皆是佛法

也許這就是我什麽我修道卻又沒有心儀的道場想入，我修的道，它不叫道，它不是道教的道。它是老子也無名的強曰“道”
一切有爲法 皆是佛，那這有之外的無呢？
法之外的虛呢
我修的道無名，他不是道，我衹不過是想不着什麽詞匯，便取用的老子之強曰“道”，也正是它確實如道這個字眼，脚下路途，道。

你是說一切法皆是佛法，无论有为无为，解脱就是佛法
法无定法
世出世间法

如果修的是佛呢？攸寧修 的是佛還是強曰爲“佛”？
是啊，不过是个名称，境至解脱都是如来
也許這就是爲什麽佛本是道了，大家都修的一樣目的，衹不過路途不同罷了。
于現階段，我這隔壁道上的外人在疑慮佛這條隔壁的路，自在麽？安然麽？
“道”是沒有等級的，我不是道教徒子
道，老子莊子列子都沒有提到神明
转念即可
立地成佛
爲什麽佛要斷七情，滅六欲？
心外无物
道的清心寡欲是在經歷中漸漸明晰什麽情什麽欲是自己本心所不求的，什麽情什麽欲是于自身本心真正看重珍惜的。
然後發現，自然自在逍遙，安然才是自己的本欲。

你說不是斩断，而是不要牵绊
要念四重恩，其中就有父母恩

這份不牽絆，感恩在口在心卻未全然在行，那深邃望透天際的目透過那孤寂處一盞青燈的迷蒙燭火，于心于口均在言“行”的感恩帶給自己的是真的解脫還是落寞？
你們這些參悟佛理的修持者，總言説著解脫、安然，卻總多是窠臼中難拔，終點才有的解脫到底還有多遠？你們都讓我心疼了
抱歉，我好像偏執了，我師父也曾勸我莫多念想他人的途。人均有自身的道，我也知我無資格干擾。

是通往光明的旅途上

是我偏執了些許，是了，心之所向，道之所存。它叫什麽，路在何方都無所謂，無論其曰佛還是名道。
也無所謂常得逍遙還是不見終途不得自在，心之所向，且望且行就是最心安自在了。
