十个国外VPS主机购买使用对比：稳定,便宜,方便的VPS主机使用评测

tydqq Post in Serveice for Net，
29 七月 2015
0

写这篇文章的目的就是为了给新手朋友们挑选国外便宜VPS主机的一个参考。

现在国外的便宜VPS如“雨后春笋”般地层出不穷，但是VPS倒闭和删除用户账号等一系列问题也是“接踵而至”，VPS市场充斥着各种陷阱，初入者如果不擦亮眼睛很可能买到失望的产品。

上一次部落写过一篇关于VPS主机的总结性的文章：七个国外便宜VPS主机使用体验，让我没有想到的排在第一位的BurstNET居然倒闭了，原本对一些老牌VPS的“坚挺”有足够的信心又瞬间凌乱了。好在BurstNET关闭前，给了用户足够长的时间来迁移数据，这也算是一个责任吧。

使用国外的VPS主机主要有两个问题：一是付款。只有部分VPS主机商是可以使用国内的支付宝付款，而多数国外的VPS主机只能使用Paypal或者信用卡来购买。二是售后问题。对于无人管理型的VPS主机，除非是宕机或者机房断网，VPS出现的其它的问题都不会过问。VPS主机影响到整个机房则可能面临关停不能恢复的风险。

这一点，对于国人来说尤其是比较看重的。因为语言和地域上的差别，国人在购买了国外的VPS主机遇到矛盾与纠纷时，往往处于一种劣势，申诉等基本上没有机会，想要追回付款只能是采用基本的办法，例如Paypal提供的争议和补偿申请就是迫于无奈下的办法：Paypal争议和补偿申请退款 。

除了VPS主机可供参考外，部落还总结了不少关于免费空间、美国主机、网赚等方面的文章，有兴趣的朋友们可以借鉴一下：

十个国外VPS主机购买使用对比：稳定,便宜,方便的VPS主机使用评测

一、Google Compute Engine谷歌云服务器平台

1、Google Compute Engine是由谷歌推出的云服务器平台，它与亚马逊Amazon AWS、微软Windows Azure应该算是国外三个有实力背景的云计算平台，GCE部落体验得不是很多，主要是因为GCE需要用信用卡才能支付，但是作为谷歌的产品还是有其强大之处。

2、经过一系列的持续开发，Google Compute Engine在数据中心、跨区域备份、SSD 硬盘、数据存储及分发等方面逐渐缩小了与Amazon AWS之间的差距，DNS解析采用Google Domains 和 Google Cloud DNS，主机机房在美国和亚太地区都有，国内连接GCE服务器速度比较快。

  • 1、Google Compute Engine云服务器激活开通和使用图文教程
  • 2、Google Compute Engine云服务器SSH管理和GCE VPS性能测试
  • 3、轻巧AMH虚拟主机面板安装使用和GCE云空间搭建网站实例

二、Linode 便宜稳定强大的美国日本新加坡VPS主机

1、如果说Google Compute Engine适合企业用户购买使用的话，那么Linode 就应该适合个人用户来购买使用了。不管是从VPS主机的价格、配置还是服务支持，Linode 都应该算是VPS界做得相当好的典型代表了，在美国、日本、新加坡等都有自己的机房，连接国内的速度快。

2、Linode VPS主机1GB内存、24GB SSD硬盘、2TB月流量，只需要10美元一个月。这个性价比即便国内的阿里云也根本无法与其相比。Linode VPS还有强大的VPS管理后台，IP绑定、数据迁移、硬盘扩容等都是可视化的操作，Linode VPS的客服支持响应速度非常快。

  • 1、10美元Linode日本美国便宜VPS主机购买和Linode VPS速度性能评测
  • 2、Linode搬家Clone克隆,IP Swap换IP,降级升级和负载均衡NodeBalancers
  • 3、删除Linode VPS主机获得退款方法和Linode好友注册奖励20美元

三、Vultr 便宜的美国、日本、欧洲等VPS主机

1、Vultr无论从VPS主机架构还是网站界面上，看得出来是想做第二个“Linode”，目前Vultr提供了美国、日本、欧洲等地区的VPS主机。Vultr与Linode相比，有两个优点：一是可以用Paypal付款，Linode只能使用信用卡来购买；二是日本机房比较充足，Linode东京机房已经有半年时间没有开卖了。

2、Vultr的VPS主机配置高价格便宜，计费也和Linode一样采用按时计费的方式，国内电信走的是电信到日本机房的快捷线路，速度还可以。Vultr支持“Custom ISO”，你可以自己上传Windows ISO来安装Windows系统，用Linux VPS的钱应用于Windows VPS上，这也是好多人愿意去购买Vultr的原因。

  • 1、Vultr便宜的日本美国欧洲VPS主机购买使用和服务器性能测评
  • 2、Vultr日本美国VPS主机性能和速度测试报告
  • 3、Vultr日本美国便宜VPS主机购买和使用方法-免费2月试用充值送美元

四、VPS.NET 专业的日本、新加坡、美国VPS主机

1、VPS.NET 是UK2Group旗下以专注提供VPS主机的公司，VPS.NET 的VPS主机的机房设立在日本、新加坡、美国等，不同的机房的VPS主机的价格也会不一样，美国和欧洲地区的VPS主机最便宜，只要15美元/月就可以购买到。

2、不过，VPS.NET 的VPS主机的性价比却远不如上面的Linode、Vultr等，就拿最便宜的一款VPS主机来说，15美元也就只能购买到512MB内存、10Gb硬盘、3TB月流量的VPS配置，该配置在新加坡、日本更是高达25美元。

  • 1、VPS.NET香港日本VPS主机$10/月购买开通过程和VPS性能测试
  • 2、VPS.NET香港日本VPS主机Paypal续费方法和几点使用感受

五、Host1plus 便宜的可用支付宝付款VPS主机

1、Host1plus 的VPS主机实在是太便宜的，便宜得不得不让人感觉得有些不放心。VPS主机最低一款内存有256MB，价格为2.5美元，可以用支付宝付款。对于Host1plus印象比较好的原因可能是Host1plus之前一直为大家提供免费VPS主机Host1Free，VPS服务还算放人信任。

2、Host1plus的产品包括虚拟主机、VPS、服务器、域名等等，机房可以选择德国、美国、巴西、非洲，OpenVZ架构，美国机房主机位于美国东海岸，所在连接国内的速度不是很快，Ping值也偏高。低配置的VPS主机适合学习测试之用。

  • 1、Host1plus便宜VPS主机用支付宝购买方法和VPS主机性能速度测试
  • 2、Host1plus虚拟主机支付宝购买年付送域名和host1plus主机速度测试

六、RamNode 便宜的OpenVZ SSD和KVM SSD VPS主机

1、RamNode的VPS主机还是以价格便宜为主，有OpenVZ SSD和KVM SSD VPS两种，最便宜的一款是3.5美元一个月，配置为内存256MB，硬盘8GB，月流量是1000Gb，在美国New York City、Atlanta、Seattle、Los Angeles等都有机房。

2、RamNode提供VPS主机也已经有了几年了，VPS主机的价格一直走的低端线路，好在VPS主机的质量还过得去，现在发展势头还可以，用户也在不断增加当中。国内用户要购买RamNode的VPS主机的话，可以使用Paypal来付款，购买后是立即开通的。

  • 1、RamNode便宜VPS主机购买开通过程和VPS性能测试结果
  • 2、RamNode便宜VPS管理使用:重装系统搭建网站和服务器

七、Virpus 老牌的VPS主机服务商

1、Virpus算是国外一批比较早提供VPS主机服务的主机商了，部落用的最早的VPS就是从Virpus开始的。从最开始的XEN VPS，到现在的SSD VPS、SSD-Cache VPS 等，可以看出Virpus一直在不断改进当中，当然Virpus还有便宜的服务器。

2、Virpus最便宜的一款SSD-Cache VPS 主机是4核心，内存1GB、SSD-Cache DISK为50GB、月流量为3TB，月付是5美元/月，支持使用Paypal付款。Virpus有时会推出一些特价的独立服务器，价格在50美元/月左右。

八、YardVPS 支持支付宝付款的便宜VPS主机

1、YardVPS据说是由几个华人开办的VPS主机服务，属于PhotonVPS旗下，VPS产品一直以低价来吸引用户，数据中心是在美国的洛杉矶。YardVPS的VPS主机都是无管理型的，提供包括XEN PV和windows系统的VPS产品。

2、YardVPS一直走低端路线，价格比较低，但是VPS主机的配置比较高，最便宜的一款VPS主机的价格是7.95美元/月，采用Xen虚拟技术，内存768 MB，硬盘30GB，月流量是1500GB，一个独立IP地址，可以使用支付宝付款。

  • 1、YardVPS美国便宜VPS终身五折支付宝$3.9/月512MB内存1.5TB月流量

九、OneAsiaHost 来自新加坡的VPS主机商

1、OneAsiaHost是一家来自新加坡的VPS主机和服务器服务商，VPS主机支持Paypal付款，加上价格比较便宜，机房位于新加坡，连接国内速度稍快，所以赢得了不少国人的喜欢。

2、关于OneAsiaHost 的评价，有好的也有不好的，主要在于OneAsiaHost 强硬的交易规则，VPS主机购买后不允许退款和付款后还得等上两天时间才可以开通VPS，稍有不慎会导致删除账号。

  • 1、新加坡OneAsiaHost购买过程和VPS性能测评附net-speeder网速优化器使用

十、Conoha.jp 日本高速便宜VPS主机

1、Conoha.jp是日本GMO集团下的以提供VPS服务器为主的网络服务商，认识Conoha.jp是因为Conoha.jp在前一段时间为大家提供免费一个月的日本VPS主机试用，VPS主机配置和体验都表现不错。1G内存2核100G硬盘，100M共享带宽，KVM架构，信用卡月付为1300日元。

2、Conoha.jp的VPS主机机房位于日本，连接国内的速度比较快，在使用Conoha.jp的VPS主机时不会出现延时的问题。Conoha.jp的 VPS主机亮点在于强大和方便的后台管理面板，安装操作系统和执行相关的命令等都可以在Conoha.jp的管理后台中完成。

  • 1、Conoha.jp日本免费试用VPS主机和VPS主机性能测试体验

国外VPS主机购买使用小结

1、其实看了上面的便宜VPS购买和使用介绍，再来对比一下阿里云最近推出的美国阿里云VPS主机，瞬间感觉阿里云的VPS主机性价比已经无足轻重了，国内的VPS主机的带宽可以按照1MB的价格来卖，美国的VPS主机敢用把带宽卖出高价的也就阿里云一个了。

2、我们购买国外VPS主机有一个非常重要的“看点”就是VPS主机的速度要好，由于地理环境的原因实际使用的过程中日本东京、香港、新加坡等亚太地区的机房连接国内的速度比较理想，这些地方的机房也是大家关注的重点，价格也会比美国地区机房的VPS要贵一些。
