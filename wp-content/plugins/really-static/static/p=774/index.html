世上並沒有什麼不能熬過

tydqq Post in 未分类，
16 三月 2015
0

世上並沒有什麼不能熬過的事物，只要一口氣在，什麼所謂絕望、心碎、煎熬都是懦弱之藉口。
不曾經歷者，亦或者略微經歷者以之說事，殊不知所謂自殺、絕食、自棄、自嘲、自責都不過是他人眼中所不屑之幼稚。這些無病呻吟也好，有病呻吟也罷的人，我們誰比誰差勁？
似乎一不小心我又感到失去了所有，但是這次我卻莫名怒了，這是什麼鬼？
並不適合這江湖？這萬丈紅塵我還非力闖一場不可，管它群雄逐鹿，人心叵測還是勾心鬥角之腥風血雨，這輩子我便就此戰個痛快。
友人花某娶她的資格有了，他的她卻沒了。
而我卻是娶她的資格尚無，我的她已經沒了。
熊熊火起，誰允許你們這些不懂珍惜的人總作矯情，不知輕重別扭鬧騰耍性子。對於一切均失只剩一病軀殘魂孤家寡人者，盡是作孽與不珍惜即視感。
到底要怎麼樣，我們這些混蛋才懂得珍惜與淡定！！！
是什麼自以為是的無私讓人斷了應珍惜之珍惜自己者斷去不甘再失之念想。
怒了這場煙塵，我就此戰個痛快。功成身就日，退隱山林時。
