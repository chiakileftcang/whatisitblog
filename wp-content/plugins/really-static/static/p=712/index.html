[轉]WordPress编辑器允许插入空格

tydqq Post in Development，
22 二月 2015
4

       為什么我在后台编辑器敲了好多空格，结果前台输出时，一点空格都没有了，在我们使用wordpress发表文章时，会遇到这些尴尬，有时费了好大劲儿在文章开头敲了空格，但在下一次编辑时，又没了，至到发布，当我们浏览时还是发现文章的开头是顶格的，在网上也看到了许多网页面对这些问题的困惑，也有许多的解决方法。好像新版的wordpress已经对这个问题进行了改善，个别空间下的还是不能达到预期效果，有的使用老版本的也在遭受这样的困境，这能说是我们不懂中文的书写规则吗，每一段都这样敲空格，而且不是敲两下就完事了，据我的经验，一般情况下至少要敲8次才能达到预期效果。

       终于找到这个好方法，当你敲多少空格它都不会自动消失，爱永设计亲自测试该方法好用，分享给大家：

1、打开当前主题模版文件夹下的functions.php文件

2、在其中添加以下代码：

//使WordPress在原生编辑器（tiny）下发表文章时，不去除空格
add_filter(‘tiny_mce_before_init’, ‘preserve_nbsp_chars’);
function preserve_nbsp_chars($initArray) {
$initArray[‘entities’] = ‘160,nbsp,’.$initArray[‘entities’];
return $initArray;
}

3、特别注意，该方法是作用在Tiny编辑器（WordPress原生编辑器）下的，如果你的WordPress的编辑器更换了，这方法也就不能用了。
