Windows系統下安裝Linux

tydqq Post in Linux，
18 四月 2015
0
    在進行以下步驟前請給硬盤留足夠的未分區空間喲。~不屬於任何拓展分區的未分區塊~

  1. 建立一個FAT32格式的分區，大小自便，至少是所需要安裝的Linux系統的ISO鏡像的兩倍~。
  2. 將Linux的ISO鏡像放入分區，並且將ISO中的images和isolinux文件夾複製一份到根目錄，以及複製一份isolinux文件夾內的initrd.img和vmlinuz文件到根目錄，如圖所示
  3. 利用EasyBCD新建Linux安裝鏡像所在分區的啟動引導
  4. 修改引導配置文件的配置信息。。。。如圖顯示，代碼是：kernel (hd0,8)/vmlinuz linux askmethod repo=hd:/dev/sda10:/initrd (hd0,8)/initrd.imgBoot(hd0,9)處的9數字是於我硬盤中那個fat32分區的物理標序，硬盤保留0~3給主分區，而我這裡如前圖是邏輯分區中的第6個，從4開始數，物理標序是9。sda10中的數字10是分區的邏輯標序，通常是之前的數字加1~
  5. 重啟進入新的啟動項進行安裝，我這裡以centos 6為例
  6. 看到這個時候按回車或者B 就可以啟動了
    當然也可能會提示找不到文件，沒關係 一定是你填寫的配置文本中分區序號填錯了。~

    將填錯的分區序列改正，選擇條目按E就能進入編輯了~kernel (hd0,8)/vmlinuz linux askmethod repo=hd:/dev/sda10:/

    initrd (hd0,8)/initrd.img

    兩個條目中的分區序號都要改過來 ，sda‘？’中數字也要相應比(hd0,?)中的數字多1。

    改完了就可以回車保存

  7. 並且按B或者回車運行進入安裝介面了。~選擇安裝語言，我們最好找到上方的Chinese(Simplified) 簡體中文。~

    確定

    選擇鍵盤樣式，默認us就ok了

    因為這裡是將硬盤分區模擬成的安裝鏡像盤，所以我們選擇Hard drive 硬盤驅動器

    選擇你的鏡像盤符，我們都知道這裡我們是sda10即邏輯上的第十個分區，所以選擇sda10（其實就算選其他的也會提示錯誤然後從新選擇的，總會選對）

    開始進入安裝引導了。

    還記得事先建議保留的未分區自由空間嗎？這裡派上用場咯。~~

    開始加系統引導項了，新手可無視，下一步

    我們選擇desktop ，千萬不要選擇minimal的喲，那個是最精簡，很坑爹的。。。可以自己選擇需要，通常我們選擇desktop就好了 有需要的server可以進入系統後再聯網安裝。在下方可以選擇現在自定義，添加需要的功能，比如網絡工具。。。。

    開始安裝了

    這時候點擊重新引導會直接重啟進入linux系統做最後用戶配置，不用擔心windows引導消失~

    如果出現這個提示，不用當心 繼續ok下一步 可以忽略，這是出現宕機時候是否有空間存放正在工作之進程，沒有也無妨，或者可以後期修改
